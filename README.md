## Beginner Modules

### BEG-00
**Module Name:** Driving Base Build

**Download:** [Driving Base Build](1_basic/BEG-00_Driving-Base-Build.pdf)

**Description:** This module involves constructing the base for the robot used in the initial learning stages of FLL.

### BEG-01
**Module Name:** Perimeter Square

**Download:** [Perimeter Square](1_basic/BEG-01_Perimeter-Square.llsp3)

**Description:** Learn to navigate the robot in a square perimeter using basic movement commands.

### BEG-02
**Module Name:** Perimeter Forward (FWD)

**Download:** [Perimeter Forward (FWD)](1_basic/BEG-02_Perimeter-FWD.llsp3)

**Description:** Practice driving the robot forward along a defined perimeter.

### BEG-03
**Module Name:** Black & White (B&W) Stop-n-Go

**Download:** [Black & White (B&W) Stop-n-Go](1_basic/BEG-03_B&W-Stop-n-Go.llsp3)

**Description:** Program the robot to stop and go based on black and white color detection.

### BEG-04
**Module Name:** Red Green Blue (RGB) Speed Variation

**Download:** [Red Green Blue (RGB) Speed Variation](1_basic/BEG-04_RGB-Speed-Variation.llsp3)

**Description:** Control the speed of the robot based on red, green, and blue color detection.

### BEG-05
**Module Name:** Perimeter Force

**Download:** [Perimeter Force](1_basic/BEG-05_Perimeter-Force.llsp3)

**Description:** Learn to navigate the robot with added force to handle obstacles.

### BEG-06
**Module Name:** Parallel Park 1

**Download:** [Parallel Park 1](1_basic/BEG-06_Parallel-Park-1.llsp3)

**Description:** Practice the basics of parallel parking the robot.

### BEG-A-T
**Module Name:** TUT: If Statements

**Download:** [TUT: If Statements](1_basic/BEG-A-T_If-Then-Statements.llsp3)

**Description:** Tutorial on using if-then statements for decision-making in programming.

### BEG-07
**Module Name:** Escape of the Walls 1

**Download:** [Escape of the Walls 1](1_basic/BEG-07_Escape-of-the-Walls-1.llsp3)

**Description:** Program the robot to escape from a boxed area using basic sensors.

### BEG-08
**Module Name:** Escape of the Walls 2

**Download:** [Escape of the Walls 2](1_basic/BEG-08_Escape-of-the-Walls-2.llsp3)

**Description:** Advanced techniques for escaping a confined area.

### BEG-09
**Module Name:** Maze 1

**Download:** [Maze 1](1_basic/BEG-09_Maze-1.llsp3)

**Description:** Navigate through a simple maze using basic movement commands.

### BEG-10
**Module Name:** Parallel Park 2

**Download:** [Parallel Park 2](1_basic/BEG-10_Parallel-Park-2.llsp3)

**Description:** Intermediate parallel parking techniques.

### BEG-11
**Module Name:** Parallel Park 3

**Download:** [Parallel Park 3](1_basic/BEG-11_Parallel-Park-3.llsp3)

**Description:** Advanced parallel parking techniques.

### BEG-12
**Module Name:** Maze 2

**Download:** [Maze 2](1_basic/BEG-12_Maze-2.llsp3)

**Description:** Advanced maze navigation techniques.

### BEG-13
**Module Name:** Freeform Maze

**Download:** [Freeform Maze](1_basic/BEG-13_Freeform-Maze.llsp3)

**Description:** Navigate through a maze with no predefined path.

### BEG-EXIT
**Module Name:** EXIT: Crane Sort

**Download:** [EXIT: Crane Sort](2_intermediate/BEG-EXIT_Crane-Sort.llsp3)

**Description:** Final project for beginners to sort objects using a crane mechanism.
